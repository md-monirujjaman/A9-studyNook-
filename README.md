# Next.js Project

A modern web application built with **Next.js** using the App Router.

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1. Install Dependencies

```bash
npm install
```

or

```bash
yarn install
```

or

```bash
pnpm install
```

or

```bash
bun install
```

### 2. Start the Development Server

```bash
npm run dev
```

or

```bash
yarn dev
```

or

```bash
pnpm dev
```

or

```bash
bun dev
```

### 3. Open Your Browser

Visit:

```
http://localhost:3000
```

Your application will be available at the above URL.

---

## 📁 Project Structure

The main page is located at:

```text
app/page.js
```

Edit this file to start developing your application. Changes are reflected automatically thanks to Hot Reload.
